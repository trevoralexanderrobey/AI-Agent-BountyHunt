# OpenClaw Bridge Runtime (AI-Agent-BountyHunt)

Local-only OpenClaw runtime stack rooted at:
- `/Users/trevorrobey/AI-Agent-BountyHunt/openclaw-bridge`

This repo runs the OpenClaw bridge, GitHub Pro MCP server, and OpenCode daemon against local Ollama.

## Current Architecture

- Local LLM: Ollama (`http://localhost:11434/v1`), default model `qwen2.5-coder:7b`
- Bridge API: OpenClaw bridge (`http://127.0.0.1:8787`)
- MCP server: `github-pro-mcp/dist/server.js`
- OpenCode daemon: `http://127.0.0.1:8091` (with internal OpenCode server on `http://127.0.0.1:8090`)
- Runtime skills loaded from: `/Users/trevorrobey/.openclaw/skills`
- Skill source-of-truth in repo: `/Users/trevorrobey/AI-Agent-BountyHunt/openclaw-bridge/skills`

## Process Ownership

Use one supervisor at a time.

- Recommended: PM2 (`openclaw-bridge`, `openclaw-mcp`, `openclaw-opencode-daemon`)
- Optional alternative: macOS LaunchAgents generated by `github-pro-mcp/scripts/mcp-install-launchd.sh`

Do not run PM2 and LaunchAgents for the same services at the same time.

## Quick Start

```bash
cd "/Users/trevorrobey/AI-Agent-BountyHunt/openclaw-bridge"
npm install
npm run bridge:build
npm --prefix github-pro-mcp install
npm --prefix github-pro-mcp run build
```

Start full stack (bridge + MCP + OpenCode daemon):

```bash
cd "/Users/trevorrobey/AI-Agent-BountyHunt/openclaw-bridge/github-pro-mcp"
./scripts/mcp-start-pm2.sh
```

Health checks:

```bash
curl -sS http://127.0.0.1:8787/health | jq
curl -sS http://127.0.0.1:8091/health | jq
curl -sS http://localhost:11434/v1/models | jq
```
