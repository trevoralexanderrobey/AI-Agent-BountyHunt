# AG Runtime For OpenClaw

This directory is the generated runtime bundle for Antigravity's OpenClaw MCP client.

Files:
- `mcp.json`: Active MCP server config Antigravity reads.
- `install-to-antigravity.sh`: Copies `mcp.json` into Antigravity User config with backup.

Install/update:
```bash
cd "/Users/trevorrobey/AI-Agent-BountyHunt/AG for OC"
./install-to-antigravity.sh
```
